1. [Section 1 Title]

On the first day at internship of tinkerers' Lab we were Introduced to Github and VScode,Today was my first day at the Tinkerers Lab. I got introduced to GitHub — basically a platform where you can store your code online and track all the changes you make to it over time. Before this I had no idea how developers manage their code professionally. We also looked at VS Code, which is the editor where you actually write your code. What was interesting is that VS Code has GitHub built right into it, so you don't need to use any separate tools. By the end of the session I learned how to push my code from VS Code to GitHub — first staging the files, then adding a commit message, and finally pushing it to the remote repository. Got it working on my own which felt great for day one..]

During the session, I created my first repository on GitHub and connected it to VS Code. I made a simple file, staged it, wrote a commit message and pushed it to the remote repository. Seeing my code actually show up live on GitHub after the push was a good feeling — that's when it really clicked for me how the whole thing works.
